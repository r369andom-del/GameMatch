# GameMatch 🎮

> AI-powered game recommendation site — find your perfect game across Mobile, PlayStation, PC and Nintendo Switch.

## Features

- 🔍 Smart game recommendations powered by AI
- 🎯 Filter by genre, mood, difficulty, art style, multiplayer, budget and more
- 🏎️ Game types: Battle Royale, Racing, Shooter, Life Sim, Survival and more
- 💾 Shows price and install size for every game
- 📚 Click any game to explore its full franchise series
- 🔄 Reset all filters with one click

## How to use

1. Select your platform (Mobile / PlayStation / PC / Nintendo Switch)
2. Pick your game type and preferences
3. Click **Find My Games**
4. Click any result card to see the full game series

## Tech

- Pure HTML, CSS, JavaScript — no framework
- AI powered by [Groq](https://groq.com) (Llama 3.3 70B)
- No backend, runs entirely in the browser
